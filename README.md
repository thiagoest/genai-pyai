# PyAI - Curso de Especialização em IA Generativa

Repositório para armazenar materiais, aulas e projetos do curso de especialização em IA generativa.

## Estrutura

- **materiais/** - Materiais do curso (slides, documentos, etc.)
- **aulas/** - Códigos e exercícios das aulas
- **projetos/** - Projetos práticos do curso
